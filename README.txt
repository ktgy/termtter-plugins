このレポジトリのファイル群は、termtter ( http://wiki.github.com/jugyo/termtter ) にて使われることを想定したプラグインです。


[bitly.rb]
http://d.hatena.ne.jp/ktgy1016/20090330/termtter_bitly
URLをbit.lyで短縮するプラグイン。

[confirm.rb]
本体添付のconfirmプラグインに、文字数カウントを付与したもの。
また、長文投稿時にpromptがおかしくなることがあった（OSX Terminal）のを回避。

[growl2.rb]
http://d.hatena.ne.jp/ktgy1016/20090329/termtter_growl
termtter本体に取り込まれているので、そちらを使うのがよいです。

[growl_kai.rb]
改造版Growl ( http://memogaki.soudesune.net/2chResView.html#rNavi6 ) のためのプラグイン。
同所にて配布されている、HTMLView.growlStyleも必要です。
テキスト中に、画像のリンクらしきものがあれば、サムネイルらしきものを引っぱってきて表示します。
現在、携帯百景、はてなフォトライフ、twitpic、YouTube（サムネイル表示）、ニコニコ動画（サムネイル表示）に対応したつもりです。

[open_url.rb]
http://d.hatena.ne.jp/ktgy1016/20090412/termtter_open_url
typable_idと連携して、テキスト中のリンクを開くプラグイン。

[outputz.rb]
http://d.hatena.ne.jp/ktgy1016/20090326/termtter_outputz
outputzプラグインがエラーを吐くことがあったのを修正した、つもり。

[retweet.rb]
typable_idと連携して、ReTweetを行うプラグイン。

[tweetburner.rb]
http://d.hatena.ne.jp/ktgy1016/20090330/termtter_tweetburner
URLをtweetburnerで短縮するプラグイン。

[typable_id.rb]
http://d.hatena.ne.jp/ktgy1016/20090412/termtter_typable_id
ID指定の補助をするプラグイン。
